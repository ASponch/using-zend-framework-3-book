7. Collecting User Input with Forms

In this chapter, you will become familiar with using web forms for gathering data entered by website users. In Zend Framework 3, functionality for working with forms is mainly spread across four components: the Zend\Form component, which allows you to build forms and contains the view helpers for rendering form elements; the Zend\Filter, Zend\Validator and Zend\InputFilter components which allow you to filter and validate user input:

Component Description
Zend\Form Contains base form model classes.
Zend\Filter Contains various filters classes.
Zend\Validator Implements various validator classes.
Zend\InputFilter Implements a container for filters/validators.
Zend\Mail Contains E-mail sending functionality.
