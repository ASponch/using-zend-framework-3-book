Предисловие

Об этой книге

Книга "Использование Zend Framework 3" - это бесплатная книга о том, как создавать веб-сайты с помощью Zend Framework 3 (или коротко ZF3). Данная книга также является open-source проектом, то есть все желающие могут участвовать в ее создании.

ZF3 - это современный PHP фреймворк, предназначенный для разработки профессионально выглядящих, масштабируемых и безопасных веб-сайтов. Такие веб-сайты легко тестировать и поддерживать. Фреймворк использует лучшие практики (best practices) и широко применяемые шаблоны проектирования (design patterns), которые были созданы в процессе развития индустрии веб разработки. Эти шаблоны включают в себя и шаблон Модель-Вид-Контроллер, позволяющий организовать код однообразно и стандартизованным способом.

Эта книга не только о Zend Framework 3, но также и о тесно связанных с ней библиотеках. Хотя Zend Framework 3 имеет собственный компонент для управления базами данных, в этой книге мы будем использовать стороннюю библиотеку под названием Doctrine ORM, которая является де-факто стандартной объектно-ориентированной библиотекой для взаимодействия с базой данных. В примерах, которые мы создадим в главах этой книги, мы будем использовать CSS фреймворк Twitter Bootstrap. Он позволяет создавать прекрасное визуальное оформление HTML элементов на ваших веб страницах.

Для кого эта книга

Эта книга предназначается для веб разработчиков, которые участвуют в создании веб сайтов на PHP. Автор стремится давать материал, начиная с простых вещей, которые должны быть понятны для начинающих. Более сложные вещи даются в последнюю очередь. Это делает данную книгу первой книгой о Zend Framework 3, понятной для новичка. Вам не нужно быть гуру в шаблонах проектирования, чтобы понимать большую часть содержимого.

Чтобы читать и понимать эту книгу, вам нужно иметь базовые представления о языке PHP. Хорошей отправной точкой для этого является его официальный веб-сайт и онлайн документация. Также будет хорошо, если вы будете иметь понимание, что такое HTTP запрос, GET и POST переменные, пространства имен (namespaces), классы и интерфейсы.

Поскольку PHP тесно связан с другими веб технологиями, также рекомендуется знать следующее:

  • HTML (Hyper Text Markup Language) -- используется для создания веб страниц, которые можно отображать в браузере.
  • CSS (Cascading Style Sheets) -- используется для задания внешнего вида страницы, таких вещей как размер шрифта или цвет фона.
  • JavaScript -- скриптовый язык, выполняемый на стороне клиента, делающий веб страницу более интерактивной.

Для изучения HTML, CSS и JavaScript, хорошая начальная точка - W3Schools Tutorials.
